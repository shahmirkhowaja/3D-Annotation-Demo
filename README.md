# 🎨 GLB Annotation Pro - Live Demo

**Created by:** Shahmir Khowaja

A high-performance 3D model annotation editor that allows users to upload GLB files, place interactive labels, and export standalone HTML experiences.

## 🌐 Live Demo

**Try it now:** [https://shahmirkhowaja.github.io/3D-Annotation-Demo/](https://shahmirkhowaja.github.io/3D-Annotation-Demo/)

## ✨ Features

- 📦 Upload and view 3D GLB models
- 🏷️ Add interactive annotations with rich text
- 🎨 Customize annotation styles and positions
- 📤 Export as standalone HTML file
- 🎮 Responsive 3D viewer with camera controls
- 🎯 Customizable initial camera views

## 🚀 How to Use

1. **Upload a Model**: Click "Choose Local File" or "Use Remote URL"
2. **Add Annotations**: Click "Add Label" button, then click on the 3D model
3. **Edit Annotations**: Click on any annotation to edit its title and description
4. **Customize**: Adjust line styles, colors, and annotation positions
5. **Export**: Click "Export HTML" to download a standalone file

## 🎯 Supported Formats

- GLB (Binary glTF) files
- Maximum file size: 50MB
